Política de Privacidad

1. Información Recopilada

Datos del Usuario: Recopilamos datos básicos de Discord, como el ID de usuario, nombre de usuario y el servidor en el que se usa el bot.


Datos de Mensajes: Para procesar comandos y brindar servicios, podemos recopilar y almacenar los mensajes enviados al bot. Esto incluye comandos y respuestas.

2. Uso de la Información

Operación del Bot: Utilizamos la información recopilada para operar, mantener y mejorar el bot Miby, y para proporcionar las funcionalidades que solicitas.

Comunicación: Podemos usar tus datos para enviar notificaciones relacionadas con el bot, como actualizaciones de servicio o cambios importantes.

3. Compartición de Datos

Con Terceros: No compartimos tu información personal con terceros, excepto cuando sea necesario para cumplir con la ley, responder a solicitudes legales, o proteger nuestros derechos y la seguridad del bot.

Excepciones: En caso de que necesitemos compartir datos con terceros para el funcionamiento del bot (como servicios de alojamiento), tomaremos medidas para proteger tu información.

4. Seguridad

Implementamos medidas de seguridad razonables para proteger tu información. Sin embargo, ninguna medida de seguridad es infalible y no podemos garantizar la protección total de tus datos.

5. Derechos del Usuario

Acceso y Eliminación: Tienes derecho a solicitar acceso a tus datos o a que se eliminen. Para hacerlo, contacta a jossxph@gmail.com.

Corrección: Puedes solicitar la corrección de información inexacta o incompleta.

6. Cambios en la Política de Privacidad

Esta Política de Privacidad puede ser actualizada periódicamente. Los cambios se publicarán en esta página y serán efectivos a partir de la fecha de publicación.

7. Contacto

Para cualquier pregunta o inquietud respecto a esta Política de Privacidad, por favor, contáctanos en jossxph@gmail.com

Todos los comandos de Miby

/setup_bot - starts the setting up process
/warn - warns any member you want
/timeout - times out any member you want
/kick - kicks any member you want
/ban - bans any member you want
/clear - allows you to delete either a set number of messages or all messages from a channel

$help - lists all the non-slash commands available
$react - reacts to the message with a random emoji
